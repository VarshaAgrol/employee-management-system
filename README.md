Employee Management System
A web-based Employee Management System developed using Django and MySQL to efficiently manage employee records, attendance, task assignments, and administrative reporting.

--> Features

- Employee registration and login authentication

- Employee record management

- Attendance tracking system

- Task assignment and monitoring

- Admin dashboard for reporting and management

- Responsive user interface using Bootstrap

--> Technologies Used

- Python

- Django

- MySQL

- HTML

- CSS

- Bootstrap

--> Project Highlights

- Designed and developed a complete employee management solution with database integration.

- Implemented secure authentication and structured data storage for employee records, attendance logs, and task management.

- Created an admin dashboard to monitor attendance and employee activities effectively.

- Improved understanding of database management, data handling, and dashboard-based reporting.

 --> Learning Outcomes

Through this project, I gained hands-on experience in:

- Database design and management

- SQL query handling

- Backend development with Django

- Frontend integration using Bootstrap

- Building data-driven applications

- Problem-solving and debugging
